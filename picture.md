## üé® Implementation Guide

### Step 1: Generate Images with ChatGPT
1. Copy each prompt exactly as written
2. Paste into ChatGPT with DALL-E access
3. Specify "photorealistic" and exact dimensions
4. Download the generated image

### Step 2: Find Icons
1. Visit recommended icon libraries:
   - [Heroicons](https://heroicons.com/) - Free, MIT license
   - [Lucide](https://lucide.dev/) - Free, ISC license
   - [Iconoir](https://iconoir.com/) - Free, MIT license
   - [Phosphor Icons](https://phosphoricons.com/) - Free, MIT license

2. Search for keywords provided in each icon section
3. Download as SVG
4. Change color to primary green (#2d5f3f)
5. Ensure 2px stroke weight for consistency

### Step 3: Optimize Images
```bash
# For photos (convert to WebP)
cwebp -q 85 input.png -o output.webp

# For icons (optimize SVG)
svgo input.svg -o output.svg
```
