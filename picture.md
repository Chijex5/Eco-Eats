## üé® General Recommendations

### Image Style Guidelines
1. **Consistency**: All images should have similar color grading (warm, welcoming tones)
2. **Diversity**: Show people of different ages, ethnicities, and backgrounds
3. **Authenticity**: Use realistic photos rather than overly staged stock photos
4. **Quality**: Minimum 72 DPI for web, optimized file sizes (use WebP format when possible)
5. **Accessibility**: Ensure adequate contrast for overlaid text

### Overlay Best Practices
- **Dark overlays**: Use for hero sections where text needs high contrast
- **Light overlays**: Use for decorative backgrounds behind content cards
- **Gradient overlays**: Transition from opaque to transparent for natural blending
- **Color**: Use brand colors (primary green) in overlay gradients where appropriate

### Priority Order for Implementation
1. **High Priority** (Most Impact):
   - Hero section backgrounds (Home, Learn, Spotlight, Contact)
   - "Ways to Get Involved" card backgrounds on Contact page
   
2. **Medium Priority**:
   - Success stories backgrounds on Learn page
   - Past heroes backgrounds on Spotlight page
   - Action section cards on Learn page

3. **Low Priority** (Nice to Have):
   - Pattern backgrounds
   - Decorative elements
   - Icon sets

### File Naming Convention
- Use kebab-case: `food-rescue-volunteers.jpg`
- Include size indicator: `hero-background-1920x1080.jpg`
- Use descriptive names: `lagos-food-partner-card-bg.jpg`

### Storage Location
- Place all images in `/public/images/` directory
- Create subdirectories for organization:
  - `/public/images/heroes/` - Hero section backgrounds
  - `/public/images/cards/` - Card background images
  - `/public/images/icons/` - Icon sets
  - `/public/images/patterns/` - Pattern backgrounds

## ‚úÖ Implementation Notes

After generating images with ChatGPT or other AI tools:

2. Convert to WebP format for better performance
3. Provide fallback formats (JPEG/PNG) for older browsers
4. Update alt text for all images for accessibility
5. Test on different screen sizes to ensure overlays work properly
6. Verify that text remains readable over all background images
